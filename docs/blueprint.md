# **App Name**: Skillforge Prime

## Core Features:

- Adaptive Persona Engine: Dynamically adjusts content based on Skills Graph, Karmic Alignment, and Profession Tag, personalizing the learning experience.
- Content Factory: Generates Competency-Building Events (CBEs) using the Gemini API, providing both academic and quest-based objectives.
- Dual-Path Render Engine: Renders CBEs in two modes: a traditional LMS view (Credential Track) and an RPG view (Skillforge Realm), using the same underlying data.
- AI Tutor (Talos Agent): Uses RAG against the ChromaDB to provide personalized tutoring and instant feedback, adapting to the user's skill gaps and learning style; may act as tool use to find information for the student.
- NPC Free Will Engine: AI-driven NPCs teach and mentor users, with their behavior and guidance influenced by the user's Karmic Alignment.
- Immersive Training Environment: Supports AR/VR and haptic integration for realistic simulation training, providing real-time coaching during VR tasks.
- gRPC and MCP Implementation: Utilizes gRPC for high-performance internal communication and the Model Context Protocol (MCP) for efficient AI service integration.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke feelings of knowledge, contemplation, and sophistication.
- Background color: Very light indigo (#F0F8FF), to allow the content and deeper primary tones to stand out.
- Accent color: Violet (#8A2BE2), a vivid analogous color used to guide the user towards calls to action and key information.
- Body font: 'Inter', sans-serif. Used for most of the text.
- Headline font: 'Space Grotesk', sans-serif. Used for headlines and titles.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use icons related to skill domains and game-world aesthetics, maintaining a balance between educational and immersive visuals.
- Subtle animations to highlight changes in Skills Graph progression and Karmic Alignment, providing clear and engaging feedback to the user.
