## üõ† Getting Started

### Prerequisites

* Node.js (v18+)
* npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/The-Promethean-Society/Skillforge_prime.git](https://github.com/The-Promethean-Society/Skillforge_prime.git)
    cd Skillforge_prime
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or 
    yarn install
    ```
3.  **Set up Environment Variables:**
    Create a `.env.local` file based on `.env.example` and fill in the necessary keys (e.g., database connection, API keys).
4.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

Open your browser to `http://localhost:3000` to see the result.
